### 🚀 Featured Project

**[Stratify – Intelligent DSA Analytics Platform](https://github.com/priyamdhawan/stratify-dsa-analytics)**
*A dashboard that turns your real LeetCode activity into an honest, data-backed revision plan — no manual self-rating, no invented "AI readiness scores."*

- Ingests live LeetCode data and clusters your topic-wise strengths using K-Means — tiered relative to your own profile, not a fixed benchmark.
- Tracks real historical progress in SQLite and forecasts future growth with linear regression.
- Recommends actual unsolved problems for your weak areas via the LeetCode API, prioritized by interview frequency.
